# SET/5 – GitHub Pages v1.5

## Deployment

Den **Inhalt dieses Ordners** direkt in die Root-Ebene des GitHub-Repositories laden.

Danach:

1. GitHub → Settings → Pages
2. Source: Deploy from a branch
3. Branch: main
4. Folder: /(root)

## iPhone

Safari → GitHub-Pages-Adresse → Teilen → Zum Home-Bildschirm.

## Cache
