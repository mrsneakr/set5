# SET/5 – GitHub Pages Release 1.0

Den **Inhalt dieses Ordners** direkt in die Root-Ebene des Repositories hochladen.

Danach:

1. GitHub → Settings → Pages
2. Source: Deploy from a branch
3. Branch: main
4. Folder: /(root)

Auf iPhone:

1. GitHub-Pages-Adresse in Safari öffnen
2. Teilen
3. Zum Home-Bildschirm

Nach dem ersten vollständigen Laden funktioniert die App auch offline.
