# SET/5 v0.9 auf GitHub Pages

Den **Inhalt dieses Ordners** in die Root-Ebene des Repositories hochladen.

Danach:

1. GitHub → Settings → Pages
2. Source: Deploy from a branch
3. Branch: main
4. Folder: /(root)

Auf iPhone:

1. GitHub-Pages-Adresse in Safari öffnen
2. Teilen
3. Zum Home-Bildschirm

Nach dem ersten Laden arbeitet die App offline.

## Updates

Der Service Worker verwendet den Cache `set5-v0.9.0`.
Bei einer späteren Version sollte der Cache-Name geändert werden.
